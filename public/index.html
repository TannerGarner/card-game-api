Trading Card API

Domains

Add these filters to the url to see various sets of data

See all Data like this: /cards

Filter by Set like this: /cards?set=Base%20Set

Filter by Type like this: /cards?type=Creature

Filter by Rarity like this: /cards?rarity=Common

View a single card like this: /cards/(cardId)

Admin Commands

To login as an Admin, visit /login.html

Add a new card like this: /cards/create

Update a card like this: /cards/(cardId)/update

Delete a card like this: /cards/(cardId)/delete
